# Python Basics Curriculum

A set of small, silly exercises for learning Python from scratch. No prior
Python needed - just do them roughly in order, top to bottom. Each one
builds on the constructs from the ones before it.

Every exercise folder has the same shape:

- `README.md` - the story and the task
- `solution.py` - a function with `TODO`s for you to fill in
- `test_solution.py` - run with `python test_solution.py` to check your work

## Curriculum

### 1. Variables & Output
Storing values and printing/formatting them.
1. [receipt_printer](variables_and_output/1_receipt_printer/) - print a formatted order line
2. [temperature_display](variables_and_output/2_temperature_display/) - convert and format a temperature

### 2. Conditionals
`if` / `elif` / `else`, comparisons, booleans.
1. [bubble_tea_calculator](conditionals/1_bubble_tea_calculator/) - price a drink by size and toppings
2. [arcade_ticket_booth](conditionals/2_arcade_ticket_booth/) - price a ticket by age and day

### 3. Loops
`for` and `while`, accumulator patterns.
1. [shopping_basket_total](loops/1_shopping_basket_total/) - sum a list of prices
2. [ice_cream_scoop_stacking](loops/2_ice_cream_scoop_stacking/) - count scoops with a `while` loop

### 4. Strings
Indexing, slicing, common string methods.
1. [cup_name_tag](strings/1_cup_name_tag/) - clean up a messy customer name
2. [review_vowel_counter](strings/2_review_vowel_counter/) - count vowels character by character

### 5. Lists
Building, searching, and scanning lists.
1. [topping_inventory](lists/1_topping_inventory/) - restock and check availability
2. [top_seller_finder](lists/2_top_seller_finder/) - find the best-selling item

### 6. Dictionaries
Key/value lookups.
1. [menu_price_lookup](dictionaries/1_menu_price_lookup/) - look up a price, handle missing items
2. [daily_sales_summary](dictionaries/2_daily_sales_summary/) - combine two dictionaries into a total

### 7. Functions
Parameters with defaults, and functions calling functions.
1. [bubble_tea_loyalty_discount](functions/1_bubble_tea_loyalty_discount/) - the bubble tea shop is back, now with a member discount
2. [tax_and_total_helpers](functions/2_tax_and_total_helpers/) - write one function that calls another

## Rules of the game

- Do them in order - later exercises assume you know everything before them.
- Peeking at hints in the README is fine. Peeking at an answer online defeats the point.
- Running the tests 20 times before they pass is completely normal.
